_SEnsible HeXadecimal, the ASCII-synergetic little-endian hexadecimal format._

Rust implementation of [`sehx`](https://gitlab.com/ewtoombs/sehx).

## Installation

```bash
cargo install sehx
yay -S sehx
yay -S sehx-bin
yay -S sehx-git
brew tap Fierthraix/tap
brew install --cask sehx
nix run github:Fierthraix/nur-packages#sehx
```

```powershell
scoop bucket add fierthraix https://github.com/Fierthraix/scoop-bucket
scoop install sehx
```

deb/rpm/apk/tar/zip: https://github.com/Fierthraix/sehx-rs/releases
